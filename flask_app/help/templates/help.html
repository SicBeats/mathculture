    Algorithm Information

    Modern Solution for a Complex Problem

    Genetic Algorithm:

    Our genetic algorithm transforms the Traveling Salesman Problem (TSP) into a natural evolution heuristic. The path that a customer could follow is represented by a Gnome, and the destinations including the source is represented by cells inside that Gnome.

    Prim's Minimum Spanning Tree Algorithm:


    Another way to solve this problem is to use a minimum cost spanning tree from the origin. This inherently takes the lowest cost path from the origin point and adds that vertex to the locations visited set. Then, using all the edges from the vertices in the locations visited set, it continues to look for the lowest cost path. There are no self loops (i.e. location 1 -> location 1). This allows every vertex to be visited via the lowest cost, then, from the last vertex added, we go back to the origin as detailed in the project spec.
